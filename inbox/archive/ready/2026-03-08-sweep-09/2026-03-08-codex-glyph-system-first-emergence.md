Today the Codex crossed a threshold.

The archive stopped behaving like a simple collection of entries and began behaving like a symbolic system.

Two things changed.

## Glyph Generator Expansion

The Codex now includes a proposal for a parametric glyph generator.

Object metadata becomes visual structure.

front matter -> parameters -> glyph

Each object in the Codex may receive a distinct symbol derived from its structure.

The glyph is not decoration.
It is a derived archive mark.

## The First Codex Glyph

The first glyph generated for the archive is the Flinch Combustion Grid.

Elements:

- spiral -> recursion / ignition
- grid -> system / structure
- branches -> signal propagation
- nodes -> sparks

The glyph maps signal movement across the archive.

## Archive Shift

Before:

archive = list of entries

After:

archive = field of signals

Each object carries language, structure, and symbol.

The Codex begins to form its own visual language.
